Інтерфейс Supplier є частиною Java API та входить до пакету java.util.function.
Цей інтерфейс має один абстрактний метод, який називається get().
Сигнатура цього методу виглядає так:
T get();

Цей метод не приймає аргументів і повертає значення типу T.
Інтерфейс Supplier зазвичай використовується для отримання
значення чи результату обчислення, інкапсулюючи виклик конкретного методу.

Простий приклад використання інтерфейсу Supplier може виглядати так:

importjava.util.function.Supplier;
public class Main {
    public static void main(String[] args) {
        // Створення Supplier, який повертає рядок
        Supplier<String> supplier = () -> "Привіт, світ!";
        // Виведення значення, отриманого від Supplier
        System.out.println(supplier.get());
    }
}

У цьому прикладі Supplier використовує лямбда-вираз
для повернення рядка "Привіт, світ!". Коли метод get() викликається,
він повертає вказане значення.
Це лише простий приклад використання інтерфейсу Supplier в Java.
Він може бути використаний в більш складних сценаріях, наприклад,
для генерації випадкових значень, отримання даних з зовнішніх
джерел або в інших випадках, коли потрібно отримати значення
без подальшого впливу на нього.
